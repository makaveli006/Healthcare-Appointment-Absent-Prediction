# Where to Start This Project

To get started learning and using this repository, follow these steps:

1. **Read the `README.md` file**
   - Get an overview of the project, its purpose, and usage instructions.

2. **Explore `Project Notebook.ipynb`**
   - This notebook provides a guided, interactive walkthrough of the workflow, data, and modeling process.

3. **Review the `src/` directory**
   - Contains modularized Python scripts for data loading, preprocessing, feature engineering, modeling, and prediction.

4. **Check the `data/` folder**
   - Contains sample datasets and outputs to help you understand the data flow.

5. **Interested in deployment?**
   - Look at `Model Deployment.ipynb` and `deployment_assets/inference.py` for deployment-related code and instructions.

Feel free to ask for a more detailed guide or a summary of any specific file!
